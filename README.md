# CareOn Healthcare Platform

A comprehensive healthcare management platform built with Next.js and Supabase.

## Features
- Patient Management
- Doctor Appointments
- Lab Bookings
- Health Tracking
- Reminders & Notifications
- Admin Dashboard

## Tech Stack
- Next.js 16
- Supabase
- TailwindCSS
- Lucide Icons

## Deployment
Deployed on Vercel: https://careon-indol.vercel.app
